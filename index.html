API Reference
Command Reference

All commands are PySimverse-compatible. Start with from pysimverse import Drone

drone.takeoff(height=3)
Arm and ascend to given height in meters
drone.land()
Descend and disarm motors safely
drone.hover(seconds=1)
Hold position for a number of seconds
drone.return_home()
Fly back to takeoff point
drone.move_forward(distance, speed)
Move forward relative to drone heading
drone.move_backward(distance, speed)
Move backward relative to drone heading
drone.move_left(distance, speed)
Strafe left relative to drone heading
drone.move_right(distance, speed)
Strafe right relative to drone heading
drone.move_up(distance)
Gain altitude by given distance
drone.move_down(distance)
Lose altitude by given distance
drone.go_to(x, y, z, speed)
Fly to absolute world coordinates
drone.rotate_cw(degrees=90)
Rotate clockwise by degrees
drone.rotate_ccw(degrees=90)
Rotate counter-clockwise by degrees
drone.set_speed(speed)
Set default movement speed (0.1–5)
drone.flip(direction)
Perform a flip: forward, backward, left, right
drone.orbit(cx, cy, radius, laps)
Orbit around a world point
drone.spiral(radius, height, turns)
Fly an ascending spiral pattern
drone.figure_8(size)
Figure-8 flight pattern
drone.square(size)
Fly a square patrol pattern
drone.patrol(waypoints)
Visit list of [x,y,z] waypoints in order
drone.detect_face()
Simulates face detection, returns dict
drone.detect_gesture()
Simulates gesture detection
drone.follow_line(path, height)
Follow a 2D path of waypoints
drone.get_position()
Returns [x, y, z] current position
drone.get_battery()
Returns battery % (0-100)
drone.get_altitude()
Returns current Z altitude in meters
