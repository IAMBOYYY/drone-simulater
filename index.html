API Reference
Command Reference

All commands are PySimverse-compatible. Start with from pysimverse import Drone

drone.takeoff(height=3)
Arm and ascend to height in metres
drone.land()
Descend and disarm safely
drone.hover(seconds=1)
Hold position for N seconds
drone.return_home()
Fly back to launch point
drone.move_forward(distance, speed)
Move forward relative to heading
drone.move_backward(distance, speed)
Move backward relative to heading
drone.move_left(distance, speed)
Strafe left relative to heading
drone.move_right(distance, speed)
Strafe right relative to heading
drone.move_up(distance)
Gain altitude
drone.move_down(distance)
Lose altitude
drone.go_to(x, y, z)
Fly to absolute world coordinates
drone.rotate_cw(degrees=90)
Rotate clockwise
drone.rotate_ccw(degrees=90)
Rotate counter-clockwise
drone.set_speed(speed)
Set movement speed 0.1–5
drone.flip(direction)
Flip: forward / backward / left / right
drone.orbit(cx, cy, radius, laps)
Orbit around a world point
drone.spiral(radius, height, turns)
Ascending spiral pattern
drone.figure_8(size)
Figure-8 flight pattern
drone.square(size)
Square patrol pattern
drone.patrol(waypoints)
Visit list of [x,y,z] waypoints
drone.rc_for(t, y, p, r, secs)
🕹️ RAW RC: throttle/yaw/pitch/roll -1.0 to 1.0 for N seconds
drone.detect_face()
Simulated face detection → dict
drone.detect_gesture()
Simulated gesture detection → str
drone.follow_line(path, height)
Follow 2D waypoint path
drone.get_position()
Returns [x, y, z]
drone.get_battery()
Returns battery %
drone.get_altitude()
Returns current altitude
