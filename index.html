Command Reference

All commands are PySimverse-compatible. Use from pysimverse import Drone

drone.takeoff(height=3)
Take off to specified altitude in meters
drone.land()
Land at current position
drone.hover(seconds=1)
Hold position for given seconds
drone.return_home()
Fly back to origin (0,0)
drone.move_forward(distance, speed)
Move forward relative to drone heading
drone.move_backward(distance, speed)
Move backward relative to heading
drone.move_left(distance, speed)
Strafe left
drone.move_right(distance, speed)
Strafe right
drone.move_up(distance)
Ascend by distance meters
drone.move_down(distance)
Descend by distance meters
drone.go_to(x, y, z, speed)
Fly to absolute coordinates
drone.rotate_cw(degrees=90)
Rotate clockwise by degrees
drone.rotate_ccw(degrees=90)
Rotate counter-clockwise
drone.set_speed(speed)
Set flight speed multiplier (0.1 - 5.0)
drone.flip(direction)
'forward' | 'backward' | 'left' | 'right'
drone.orbit(cx, cy, radius, laps)
Orbit around a center point
drone.spiral(radius, height, turns)
Spiral ascent pattern
drone.figure_8(size)
Figure-8 flight pattern
drone.square(size)
Fly a square pattern
drone.patrol(waypoints)
Visit list of (x,y,z) waypoints
drone.follow_line(path, height)
Follow a path of 2D points
drone.detect_face()
Returns {detected, x, y, confidence}
drone.detect_gesture()
Returns gesture string
drone.get_position()
Returns [x, y, z] coordinates
drone.get_battery()
Returns battery percentage
drone.get_altitude()
Returns current altitude
